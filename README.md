# SAP CAP HR Platform

Projet portfolio pour apprendre SAP CAP Node.js.

Objectif : construire une application RH avec CDS, services OData, SAP HANA Cloud, sécurité XSUAA et Fiori Elements.

## Roadmap

- Setup projet
- Modèle Employees / Departments / Contracts
- Services OData
- Logique métier Node.js
- Sécurité
- Déploiement SAP BTP
- UI Fiori Elements


## Concepts appris

### CAP

CAP (cloud application programme) est un framwork SAP qui sert a devlopper des application metier Cloud pour SAP en utilisant NODE.JS (express.js) ou java (java Spring)
 
### CDS
CDS (Core data service) sert a decrire des donnes ou service

### OData

Odata sert a exposer les donner en API standard et les integrer pour SAP fiori
