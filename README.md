# SAP CAP HR Platform

Projet portfolio pour apprendre SAP CAP Node.js.

Objectif : construire une application RH avec CDS, services OData, SAP HANA Cloud, sécurité XSUAA et Fiori Elements.

## Roadmap

- Setup projet
- Modèle Employees / Departments / Contracts
- Services OData
- Logique métier Node.js
- Sécurité
- Déploiement SAP BTP
- UI Fiori Elements
